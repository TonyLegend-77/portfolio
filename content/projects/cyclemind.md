## CycleMind - AI Portfolio Rebalancer

**Live Demo:** [cyclemind.vercel.app](https://cyclemind.vercel.app)

### Overview
CycleMind is an AI-powered portfolio rebalancer designed for the Bitget exchange. Built for Bitget Hackathon, it combines institutional-grade analytics with AI-driven insights to optimize trading strategies.

### My Role
**Founder & Developer**

### Key Features
- 📊 **Dashboard** - Real-time portfolio analytics
- 📈 **Perps Interface** - Perpetual futures management
- 🔍 **Arb Scanner** - Arbitrage opportunity detection
- 🛡️ **Guardian Mode** - AI risk management
- 📡 **Market Pulse** - Real-time market analysis
- 🎮 **Control Room** - Advanced trading controls

### Technology Stack

**Frontend**
- HTML/CSS/JavaScript (Vanilla)
- Multi-page architecture (index.html, perps.html, arb.html)
- localStorage-based authentication
- Institutional terminal aesthetic with SVG icons
- Deployed on Vercel

**Backend**
- Python 3.12
- FastAPI
- Deployed on Railway

**Integrations**
- Bitget API (trading data & perpetuals)
- Claude Sonnet (AI agent logic)

### What Makes It Special
✨ Built for both humans and AI agents
✨ Institutional-grade interface
✨ Real-time AI-powered rebalancing suggestions
✨ Seamless Bitget integration
