CycleMind

AI Portfolio Rebalancer for Bitget Exchange

Status: Active Development | Year: 2024

Live Demo: cyclemind.vercel.app →

Overview

CycleMind is an AI-powered portfolio rebalancer designed for the Bitget exchange. Built for Bitget Hackathon, it combines institutional-grade analytics with AI-driven insights to optimize trading strategies. The platform serves both individual traders and institutional investors who need real-time portfolio optimization.

My Role

Founder & Developer - Responsible for full-stack development, architecture design, and AI integration.

🎯 Key Features

  • 📊 Dashboard - Real-time portfolio analytics and performance metrics
  • 📈 Perps Interface - Perpetual futures management with advanced controls
  • 🔍 Arb Scanner - Arbitrage opportunity detection across multiple pairs
  • 🛡️ Guardian Mode - AI-powered risk management and position sizing
  • 📡 Market Pulse - Real-time market analysis and trend detection
  • 🎮 Control Room - Advanced trading controls and automation

Technology Stack

Frontend

  • HTML/CSS/JavaScript (Vanilla - no framework overhead)
  • Multi-page architecture (index.html, perps.html, arb.html)
  • localStorage-based authentication across pages
  • Institutional terminal aesthetic with custom SVG icons
  • Deployed on Vercel with auto-scaling

Backend

  • Python 3.12 (pinned for pydantic compatibility)
  • FastAPI for high-performance API endpoints
  • Deployed on Railway with auto-scaling
  • Async/await for concurrent request handling

Integrations

  • Bitget API (trading data, perpetuals, account management)
  • Claude Sonnet (AI agent logic for strategy recommendations)
  • Real-time WebSocket connections for market data

What Makes It Special

  • ✨ Built for both humans and AI agents
  • ✨ Institutional-grade interface that's intuitive
  • ✨ Real-time AI-powered rebalancing suggestions
  • ✨ Seamless Bitget integration - no bridging needed
  • ✨ Guardian Mode protects against large losses
  • ✨ Lightning-fast performance with vanilla JavaScript

How It Works

  1. Connect Wallet: Securely link your Bitget account via API
  2. Set Parameters: Define your risk tolerance and goals
  3. AI Analysis: Claude Sonnet analyzes your portfolio
  4. Recommendations: Get AI-powered rebalancing suggestions
  5. Execute: One-click execution of trades
  6. Monitor: Real-time tracking and adjustments

Future Roadmap

  • 🎯 Multi-exchange support (Bybit, OKX, Deribit)
  • 🎯 Advanced ML models for prediction
  • 🎯 Social trading features
  • 🎯 Mobile app for iOS/Android
  • 🎯 Advanced reporting and tax integration

Overview

Bitedge is a sophisticated AI backtest engine designed for both human traders and autonomous AI agents. It enables rapid testing of trading strategies with real market data and AI-driven optimizations. Perfect for validating ideas before deploying capital.

My Role

Developer & Founder - Full-stack development with focus on performance and accuracy.

  • 📊 Advanced Backtesting - Multi-asset, multi-timeframe testing
  • 🤖 AI Optimization - ML-powered strategy optimization
  • 📈 Perps Engine - Futures and perpetuals testing
  • 🔄 Arb Scanner - Arbitrage opportunity backtesting
  • 📡 Market Pulse - Historical market analysis
  • 🛡️ Guardian Mode - Risk management simulation

Technology Stack

Frontend

  • HTML/CSS/JavaScript (Vanilla)
  • Multi-page architecture
  • localStorage authentication
  • Institutional UI/UX
  • Deployed on Vercel

Backend

  • Python 3.12
  • FastAPI with async support
  • Deployed on Railway
  • NumPy/Pandas for data processing

Integrations

  • Bitget API (market data)

What Makes It Special
